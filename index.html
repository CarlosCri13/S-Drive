¿Cómo funciona nuestro sistema?

Tecnología avanzada para mantenerte alerta y seguro

1. Captura de imágenes

El sistema accede a la cámara del dispositivo para capturar imágenes del usuario en tiempo real.

2. Detección facial

Identifica puntos clave del rostro (ojos, boca, cabeza) usando algoritmos de visión por computadora.

3. Análisis de señales

Calcula métricas como EAR (Eye Aspect Ratio) y MAR (Mouth Aspect Ratio) para detectar fatiga.

4. Alertas preventivas

Emite notificaciones cuando detecta patrones de somnolencia que superan los umbrales de seguridad.
